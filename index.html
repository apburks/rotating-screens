Welcome to Rotating-screens

This page will give you a short demo and explanation of what this project aims to do. Any feedbacks would be greatly appreciated!

Please follow https://github.com/dandb/rotating-screens to go to the actual repository

Demo

Launch your local project and login with your default credentials (admin@admin.com/password)

Create your users as needed. Two different account types are available: Admin and user. An admin can create/edit new and current users. Admin is also allowed to edit any offices' dashboards while a user can only edit their office's dashboard and display it.

Admin Dashboard

Currently Dun & Bradstreet Credibility Corp has six offices but you may change your office locations as you desire.

User Dashboard

Click "Start Dashboard" to display your dashboard!

Each url comes with a cover page using its description as its title.

This project aims to increase the productivity of your office. You may and should change anything to fit your office's needs! Rotating-screens is in a constant development, feel free to give us any inputs and feedbacks, we greatly appreciate them.
